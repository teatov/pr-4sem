\par Научно-исследовательская практика по теме <<Обработка больших данных. Распознавание объектов на цифровых изображениях>> проходила с 22 июля по 3 августа 2024 года в Департаменте математического и компьютерного моделирования ДВФУ.
\par Цели:
\begin{itemize}
    \item изучение современных методов решения задач распознавания цифровых изображений с помощью машинного обучения.
\end{itemize}
\par Задачи:
\begin{itemize}
    \item ознакомление с основами построения и возможностями применения нейронных сетей;
    \item получение и систематизация знаний о возможностях и особенностях построения и применения нейрокомпьютерных алгоритмов и систем для обработки цифровых изображений.
\end{itemize}
\par Структура и содержание практики:
\begin{enumerate}
    \item Организационный этап. Выдача индивидуальных заданий на проведение отдельных этапов работы в соответствии с темой. Составление плана работы.
    \item Ознакомительный этап. Ознакомление с организацией работы с нейронными сетями. Изучение технологии обработки информации. Изучение прикладного программного обеспечения, используемого для работы с машинными обучением. Ознакомление с методикой работы, выбор необходимых или разработка новых методов исследования. Работа с литературными источниками.
    \item Основной этап. Изучение методов машинного обучения и создания нейронных сетей. Работа согласно индивидуальному плану. Приобретение навыков работы с библиотеками для обработки данных для нейронных сетей. Реализация обучения модели нейронной сети для распознавание изображений. Тестирование результата. Посещение научных семинаров, изучение специальной литературы по выбранной теме.
    \item Завершающий этап. Подготовка отчета по практике (систематизация результатов исследования). Утверждение отчета руководителем практики. Защита отчета по практике.
\end{enumerate}
